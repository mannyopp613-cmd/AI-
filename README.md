# 🧬 ChemiSmart Pro: The Intelligent Chemistry Mentor

**Revolutionizing Science Education with Multi-Modal AI & Real-Time Data.**

ChemiSmart Pro is a high-performance pedagogical agent designed to bridge the gap between abstract chemistry concepts and student mastery. Built for the modern learner, it combines real-time scientific grounding with dynamic progress tracking.

---

## 🚀 The Vision
Most AI tutors simply give answers. **ChemiSmart Pro teaches.** It follows a rigorous 4-step pedagogical model:
1. **Opening**: Establishing context and goals.
2. **Diagnosis**: Identifying specific misconceptions before explaining.
3. **Surgical Answer**: Providing precise, high-impact information.
4. **Proactive Step**: Guiding the learner to the next logical milestone.

## 🛠️ Key Features
- **Cognitive Mastery Map (MOC)**: A real-time dashboard that tracks the student's "State of Knowledge" across core chemistry modules.
- **Scientific Grounding**: Leveraging Google Search grounding via Gemini to provide the most up-to-date and accurate scientific facts.
- **Molecular Painter**: Generates high-fidelity 3D illustrations of molecules and laboratory setups on demand using Gemini 2.5 Flash Image.
- **Adaptive Learning Profiles**: The system identifies the student's learning style (visual, analytical, etc.) and adjusts its tone and content accordingly.

## 💻 Tech Stack
- **AI Engine**: Google Gemini 3 Flash (Reasoning) & Gemini 2.5 Flash Image (Visuals).
- **Frontend**: React 19, TypeScript, Tailwind CSS.
- **Grounding**: Google Search Grounding for real-time verification and source citations.
- **Styling**: Cyberpunk-inspired "Glassmorphism" UI for an immersive learning experience.
